# API

---

### Users

- Get all the users  (/)
- Get a user         (/i/:id)
- Get user by name   (/u/:username)
- Add a new user     (/add)
- Remove a user      (/del)


### Status

- Get all status (/)
- Get status by Id (/:id)
- Add new status (/add)
- Remove a status (/del/:id)
- Update a status: color or name (/update/:id)


### TodoGroup

- Get all groups (/)
- Get group by id (/i/:id)
- Get group by name (/n/:name)
- Add new group (/add)
- Remove a group (/del/:id)
- Update a group (/update/:id)
